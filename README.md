# broadcast-media-attribution-case-studies

Verified, evidence-based case studies demonstrating the measurable impact of broadcast media advertising (radio and television) using rigorous attribution and measurement methodologies.

---

## Overview

This repository contains empirical, data-backed case studies documenting the measurable business outcomes generated by broadcast media advertising campaigns.

Each case study includes:

- Campaign structure and duration  
- Investment levels  
- Measurement methodology  
- Third-party data sources  
- Observed performance lift across key metrics such as foot traffic, website traffic, revenue attribution, and brand awareness  
- Limitations and methodological notes for transparency

The goal of this repository is to provide transparent, evidence-based documentation of how broadcast media campaigns drive measurable business outcomes.
